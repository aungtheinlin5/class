Quick Guide

1. Books tab ကနေ စာအုပ်အသစ်ထည့်ပါ (Free / Premium ရွေးလို့ရပါတယ်)။
2. User က Website ထဲမှာ Register/Login လုပ်ပြီး "Premium ဝင်မယ်" ကိုနှိပ်ပြီး Payment Screenshot နဲ့ Order တင်လိုက်ရင် Telegram ဆီ ချက်ချင်း အကြောင်းကြားချက် ရောက်ပါလိမ့်မယ်။
3. Premium Orders tab ထဲမှာ Order ကို ကြည့်ပြီး Approve နှိပ်ပေးပါ — ချက်ချင်း Unlock ဖြစ်ပါလိမ့်မယ်။ (ဒီအကောင့်နဲ့ Browser/Device မည်သည့်ခုကနေ Login ဝင်ဝင် Premium ရနေပါမယ်)

2D ဂဏန်းလက်ဆောင် (Free/VIP)
4. 2D ဂဏန်းလက်ဆောင် tab ကနေ Post အသစ်ထည့်ပြီး Free / VIP ရွေးပါ။
5. User က 2D Live → VIP tab ထဲမှာ Register/Login လုပ်ပြီး "VIP ဝင်မယ်" ကိုနှိပ်ပြီး Payment Screenshot နဲ့ Order တင်ပါလိမ့်မယ်။
6. VIP Orders (2D) tab ထဲမှာ Order ကို ကြည့်ပြီး Approve နှိပ်ပေးပါ — VIP Post များ ချက်ချင်း Unlock ဖြစ်ပါလိမ့်မယ်။

ဈေးနှုန်း / Payment Number
7. Settings tab ထဲမှာ Premium/VIP ဈေးနှုန်းနှင့် KBZ Pay / Wave Pay name+number ကို ပြောင်းလို့ရပါတယ် — User Form ထဲမှာ ချက်ချင်း update ဖြစ်ပါလိမ့်မယ်။
