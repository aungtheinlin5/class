Quick Guide

1. Books tab ကနေ စာအုပ်အသစ်ထည့်ပါ (Free / Premium ရွေးလို့ရပါတယ်)။
2. User က Website ထဲမှာ Register/Login လုပ်ပြီး Telegram ကနေ Admin ကို User ID ပို့လာပါလိမ့်မယ်။
3. Users tab ထဲမှာ User ID ကို ရှာပြီး Approve နှိပ်ပေးပါ — ချက်ချင်း Unlock ဖြစ်ပါလိမ့်မယ်။ (ဒီအကောင့်နဲ့ Browser/Device မည်သည့်ခုကနေ Login ဝင်ဝင် Premium ရနေပါမယ်)
